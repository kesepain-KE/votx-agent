# 文件操作

提供安全的文件系统操作，所有路径受沙箱保护。

## 工具

| 工具 | 用途 |
|------|------|
| `read_file` | 读取文件内容 |
| `write_file` | 将内容写入文件 |
| `list_dir` | 列出目录内容 |
| `delete_file` | 删除文件（禁止删目录） |

## 安全约束

- 所有路径限定在用户目录 (`KESEPAIN_USER_DIR`) 和项目根目录内
- write_file 越权时自动回退到用户目录下的 basename
- delete_file 拒绝删除目录
