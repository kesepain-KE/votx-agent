# 工具调用原理

## 一、工具注册体系

### 两种 Skill 类型

| 类型 | 目录 | 注册方式 | 调用方式 |
|------|------|----------|----------|
| 工具型 | `plugins/<name>/SKILL.md` + `tool.py` | `tool.py` 中定义 schema + handler | LLM function call |
| 指令型 | `plugins/<name>/SKILL.md` 或 `skills/<name>/SKILL.md` | `SKILL.md` 注入摘要到 system prompt | LLM 按指令用已有工具操作 |

### 工具型 Skill 注册

```python
# plugins/xxx/tool.py
TOOLS = [
    {
        "type": "function",
        "function": {
            "name": "tool_name",
            "description": "工具描述",
            "parameters": {
                "type": "object",
                "properties": {
                    "param1": {"type": "string", "description": "..."}
                },
                "required": ["param1"]
            }
        }
    }
]

def handle(tool_name: str, arguments: dict, **kwargs) -> str:
    # 执行逻辑
    return "结果文本"
```

## 二、工具调用流程

### LLM 返回 tool_calls

```json
{
  "role": "assistant",
  "content": null,
  "tool_calls": [
    {
      "id": "call_abc123",
      "type": "function",
      "function": {
        "name": "read_file",
        "arguments": "{\"path\": \"config/config_soul.md\"}"
      }
    }
  ]
}
```

### ToolRunner 执行

```python
# run/tool_runner.py
class ToolRunner:
    def run(self, tool_call: dict) -> str:
        name = tool_call["function"]["name"]
        arguments = json.loads(tool_call["function"]["arguments"])
        # 1. 权限检查
        # 2. 安全检查（shell 命令危险检测、路径沙箱、SSRF 防护等）
        # 3. 超时控制
        result = handler(name, arguments, **context)
        return result
```

### 完整调用循环（run_chat_turn）

```python
# run/engine.py
def run_chat_turn(chat, tool_runner, provider, tools):
    for round in range(MAX_ROUNDS):  # 最多 10 轮工具调用
        response = provider.respond(chat.build_messages(), tools=tools)
        if response.text:
            yield {"type": "text", "content": response.text}
        if response.tool_calls:
            for tc in response.tool_calls:
                yield {"type": "tool_call", "line": f"🔧 {tc.name}({tc.arguments})"}
                result = tool_runner.run(tc)
                chat.add_tool_result(tc.id, result)
                yield {"type": "tool_result", "content": result}
        else:
            break
```

## 三、上下文绑定机制

每次对话开始前，`AgentService.chat()` 或 `_run_turn()` 会调用：
```python
ai_tool.set_auto_improve_context(provider=provider, chat=chat, user_name=username)
tp_tool.set_task_plan_context(provider=provider, chat=chat, user_name=username)
set_multimodal_context(provider=provider, chat=chat, user_name=username)
```

## 四、工具调用限制

- 每轮对话最多 **10 轮**工具调用循环
- 单次工具执行超时：默认 **120 秒**
- 工具结果过大时自动截断
- 超过限制时 yield `{"type": "max_rounds"}` 或 `{"type": "error"}`
