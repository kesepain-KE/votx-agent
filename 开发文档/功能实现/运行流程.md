# 运行流程

最后更新：2026-05-04

## CLI 流程

```
python start.py
  → 扫描 users/ → 用户选择 → 设置 VOTX_USER_DIR
  → subprocess 启动 main.py → 加载配置
  → 初始化 Provider / ToolRunner / ChatManager
  → build_system_prompt → load_tool_schemas
  → input 循环 → run_chat_turn → 保存历史和日志
```

## Web 流程

```
python start.py --web 或 python start_web.py
  → web/server.py 创建 Flask app → 导入 web/routes/*
  → 浏览器访问 → GET /api/users → POST /api/select-user
  → _init_session → POST /api/chat → run_chat_turn
  → SSE 返回事件 → Vue 3 前端渲染 → 保存历史和日志
```

## system prompt 构建

```
build_system_prompt(root, user_dir)
  → 1. self_soul.md        (用户人设)
  → 2. config/soul.md      (全局人格基座)
  → 3. AGENTS.md            (编码手册)
  → 4. register_all()      (Skill 摘要)
  → 5. agent_memory.db     (持久记忆, LIMIT 20)
  → 6. .learnings/         (学习/错误/功能请求概要)
```

注意: `build_system_prompt()` 内部调用 `register_all()`，必须先于 `load_tool_schemas()` 调用。

## 对话循环 (tool calling)

```
chat.build_messages
provider.chat / chat_stream
  → usage → thinking → text chunk
  → tool_calls → 提取 reasoning_content
  → chat.add_tool_call_message(tool_calls, reasoning_content)
  → ToolRunner.execute → chat.add_tool_results
  → 循环 (max 20 轮)
  → 无 tool_calls → 最终回复 (含 reasoning)
```

关键: DeepSeek thinking 模式下，返回的 `reasoning_content` 必须在下一轮请求中回传，否则 API 报 400。

## 工具执行

```
ToolRunner.execute
  → deny / enabled 检查
  → 全局和单工具限流
  → JSON 参数解析
  → handler 调用
  → 异常捕获转 ERROR
  → 写入 tool_log.json (每行一个 JSON)
  → 返回 tool message
```

## tool_calls 修复 (run/chat.py)

两阶段:
1. 读取历史时 — 前向+后向检查，切除孤立 tool 或未闭合 assistant tool_calls
2. 保存历史前 — 末尾未完成 tool_calls 补充错误 tool message

## SSE 事件类型

```
thinking_chunk   思考流片段
thinking_done    思考结束
thinking         非流式思考块
text_chunk       正文流片段
text_done        正文结束
text             非流式正文
tool_call        单次工具调用（含 name/args/elapsed/success）
usage            Token 用量
error            错误信息
deadlock_warning 同命令连败 3 次警告
max_rounds       达到最大工具轮数
done             对话完成
```

## 历史保存

```
chat.save_history      不含 system 的对话历史 → users/<name>/history/chat/
chat.save_log          含 system 的完整日志 → users/<name>/history/log/
chat.archive_now       归档当前对话 → users/<name>/history/archive/
summarize_and_store    生成摘要 → 写入索引
```
