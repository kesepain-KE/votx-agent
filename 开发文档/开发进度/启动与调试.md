# 启动与调试

## 安装依赖

```bash
pip install openai requests yt-dlp tavily-python python-docx
```

## 设置 API Key（4 选 1）

```bash
# 方式 1: 项目根 .env 文件（推荐）
# 编辑 .env，写入 DEEPSEEK_API_KEY=sk-your-key

# 方式 2: 环境变量
export DEEPSEEK_API_KEY="sk-your-key"

# 方式 3: 用户配置文件
# 编辑 users/<name>/config.json → provider.api_key

# 方式 4: .env.example 复制
cp .env.example .env
# 编辑 .env 填入实际 Key
```

## 启动

```bash
python start.py
```

## 斜杠命令

| 命令 | 功能 |
|------|------|
| `/exit`, `/quit`, `/q` | 退出（自动保存历史） |
| `/clear` | 清除当前对话历史（先自动归档备份） |
| `/history`, `/stats` | 查看历史状态（消息数 / 文件大小 / 归档数） |
| `/archive` | 手动归档当前历史（不清空） |
| `/help` | 显示命令帮助 |

## 退出方式

- 输入 `/exit` — 正常退出，自动保存历史
- `Ctrl+C` / `Ctrl+D` — 中断退出，自动保存历史

## 常见问题

| 问题 | 解决 |
|------|------|
| `API Key 未设置` | 创建 `.env` 文件或设置环境变量 |
| `thinking options type cannot be disabled` | 已修复：think=False 时不发 reasoning_effort |
| 工具调用无响应 | 检查 `tool_log.json` 确认是否被权限/限流拦截 |
| 热榜查询报缺少 API Key | 在 `.env` 中添加 `UAPI_API_KEY=你的密钥` |
| Shell 命令中文乱码/崩溃 | 已修复：强制 UTF-8 + errors=replace |
| 视频下载反复重试 | 确保安装了 yt-dlp: `pip install yt-dlp` |
| 工具调用超限 | 已调至 20轮/100次/单工具50次 |
| 同一命令反复重试 | 连败 3 次自动提示 LLM 换思路 |
| 路径包含 `[]` 或中文 | 用引号包裹，优先用相对路径，先 list_dir 验证 |
