# 项目概述

## 定位

多用户 AI Agent 框架。以 OpenAI 兼容接口调用 LLM，支持角色扮演、工具调用、对话持久化，内置安全机制。

## 技术栈

| 层 | 技术 |
|----|------|
| LLM | DeepSeek API (OpenAI 兼容)，可替换 |
| 语言 | Python 3.10+ |
| 依赖 | `openai` SDK, `requests` |
| 环境 | .env 文件（手动解析，不依赖 python-dotenv） |

## 当前状态

**生产可用的闭环** — 用户选择 → 加载人设+AGENT.md → 对话 → 工具调用 → Token 统计 → 命令分发 → 历史归档。

已实现:
- 核心对话循环 + 双层配置 + 历史裁剪归档
- 8 个 Skill（file / network / shell / time / uapi-hotboard-reporter / video-download / word-docx / tavily-search）共 14 个工具
- 标准 Skill 骨架（agentskills.io 规范）：SKILL.md + tool.py + 自动发现
- 8 项安全机制（错误标准化 / 路径沙箱 / 权限系统 / 限流 / 日志 / Shell 危险拦截 / SSRF 防护 / 输出截断）
- 流式输出 + Token 统计（含缓存命中显示）
- 命令系统：/exit /clear /history /archive /help
- AGENT.md 项目指南 → 注入 system_prompt，LLM 了解自身能力
- API Key 校验 + .env 自动加载
