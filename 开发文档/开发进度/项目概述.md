# 项目概述

最后更新：2026-05-04

## 定位

votx-agent 是本地多用户 AI Agent 框架。核心不是单纯聊天，而是把角色人设、全局人格基座、执行手册、Skill、持久记忆、工具调用、历史归档和 Web UI 组合成完整运行闭环。

## 已实现

- CLI: `python start.py`
- Web: `python start.py --web` (13579) / `python start_web.py` (1478, 自动轮询)
- OpenAI 兼容 Provider（DeepSeek 默认）
- `.env` 手动加载，不依赖 python-dotenv
- CLI/Web 共用 `run/engine.py`
- 20 Skill（9 工具型 + 11 指令型），25 工具
- 工具调用 + Token 统计 + 历史保存 + 日志
- tool_calls 断链修复（启动加载 + 保存前补齐）
- .learnings 记录注入 system prompt
- Web API 模块化路由（22 个端点，5 个 route 模块）
- Web 文件管理、对话归档、配置读写
- SSE 流式事件（thinking/text/tool_call/usage/error）
- 前端完整迁移至 Vue 3（CDN 加载，~2400 行）
- DeepSeek thinking 模式 tool_calls 链 reasoning_content 回传
- Web UI: 思考默认展开、AI 回复置底、工具调用横向滚动、任务执行显示开关
- Web UI: 运行状态指示 + 停止按钮、用户消息复制、发送/停止按钮位置复用
- 工具日志路径统一为 `tool_log.json`
- 全局人格基座 (`config/soul.md`) 与用户人设分离，三层职责清晰
