# 配置说明

## 全局配置 config/config_core.json

```json
{
    "history": {
        "history_max": 100,        // 最大消息条数，超限触发裁剪
        "zip_history": true        // 裁剪时归档旧消息到 history/archive/
    },
    "tool": {
        "tool_max": 100,           // 单轮最大工具调用次数
        "tool_max_per_type": 50,   // 单个工具单轮最大调用次数
        "tool_log_max": 1000,      // 工具日志条数上限（预留）
        "enabled": {}              // 全局工具开关，例 {"delete_file": false}
    }
}
```

| 字段 | 类型 | 默认 | 说明 |
|------|------|------|------|
| `history.history_max` | int | 100 | 消息条数上限 |
| `history.zip_history` | bool | true | 裁剪时是否归档到 archive/ |
| `tool.tool_max` | int | 100 | 单轮全局调用上限 |
| `tool.tool_max_per_type` | int | 50 | 单工具单轮调用上限 |
| `tool.tool_log_max` | int | 1000 | 日志上限（预留） |
| `tool.enabled` | dict | {} | 全局工具开关 |

## main.py 常量

| 常量 | 值 | 说明 |
|------|-----|------|
| `MAX_TOOL_ROUNDS` | 20 | 单轮对话最大往返次数 |

## 用户配置 users/<name>/config.json

```json
{
    "provider": {
        "name": "deepseek",
        "model": "deepseek-v4-flash",
        "api_key": "",
        "think": false,
        "stream": false,
        "timeout": 120
    },
    "history": {
        "data": "chat_data.json",
        "log": "chat_log.json"
    },
    "tool": {
        "enabled": {},
        "deny": []
    }
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `provider.name` | str | Provider 标识 |
| `provider.model` | str | 模型名 |
| `provider.api_key` | str | API Key，留空则读环境变量/.env |
| `provider.think` | bool | 是否启用 thinking |
| `provider.stream` | bool | 是否启用流式 |
| `provider.timeout` | int | 请求超时秒数（默认 120） |
| `history.data` | str | 对话历史文件名 |
| `history.log` | str | 完整日志文件名 |
| `tool.enabled` | dict | 工具开关（覆盖全局） |
| `tool.deny` | list | 工具黑名单（最高优先级） |

## AGENT.md

项目根 `AGENT.md`，启动时自动注入 system_prompt。包含：
- 项目定位、目录结构、核心架构
- 可用 Skill 清单和使用方式
- 代码约定与安全机制
- 如何添加功能

## .env 环境变量

项目根 `.env` 文件，启动时自动加载（不覆盖已有环境变量）：

```bash
# API Key（必填）
DEEPSEEK_API_KEY=sk-your-key

# 可选
DEEPSEEK_BASE_URL=https://api.deepseek.com
HTTP_TIMEOUT=15
UAPI_API_KEY=your-uapi-key
TAVILY_API_KEY=your-tavily-key
```

## 权限优先级

```
tool.deny（用户黑名单）  >  tool.enabled（用户级）  >  tool.enabled（全局）
```

## API Key 设置方式（按优先级）

1. `config.json → provider.api_key`（用户级）
2. 环境变量 `DEEPSEEK_API_KEY`
3. `.env` 文件自动加载
4. 均未设置 → 启动报错，提示设置方式
