# 配置说明

最后更新：2026-05-04

## 环境变量 (`.env`)

Provider 手动解析 `.env`，不依赖 python-dotenv。

```
DEEPSEEK_API_KEY     API 密钥
DEEPSEEK_BASE_URL    可选，覆盖默认 base_url
OPENAI_API_KEY       fallback，也用于视觉脚本
TAVILY_API_KEY       tavily-search
UAPI_API_KEY         uapi-hotboard-reporter
HTTP_TIMEOUT         network 工具超时，默认 15s
```

密钥不得写入文档、日志或回复。

## 全局配置 — `config/config_core.json`

历史限制、压缩开关、全局工具 enabled、工具调用上限、stream 默认值。

## 全局人格基座 — `config/soul.md`

注入 system prompt 的全局行为原则。定义 AI 助手的基本职业操守——诚实、安全、隐私保护。所有用户人设在此之上叠加，冲突时以用户人设优先。

## 用户配置 — `users/<name>/config.json`

```json
{
  "provider": {
    "api_key": "",
    "base_url": "",
    "model": "deepseek-v4-flash",
    "think": false,
    "stream": false,
    "timeout": 120
  },
  "history": {
    "data": "chat_data.json",
    "log": "chat_log.json"
  },
  "tool": {
    "enabled": {},
    "deny": []
  }
}
```

## 用户人设 — `users/<name>/self_soul.md`

用户专属 system prompt，定义 Agent 角色和风格（如猫猫小助手）。

## system prompt 组合顺序

```
1. users/<name>/self_soul.md    用户人设
2. config/soul.md               全局人格基座
3. AGENTS.md                     编码 Agent 手册
4. Skill 摘要                   工具型和指令型各一行
5. 持久记忆                     agent_memory.db 活跃 facts (≤20)
6. .learnings 记录              LEARNINGS/ERRORS/FEATURE_REQUESTS 概要
```

`config/soul.md` 为空或以 `<!--` 开头时跳过。

## Web 配置接口

`GET /api/config` — 读取 | `POST /api/config` — 保存（同步更新内存 provider）

## 数据库路径

- 对话历史: `users/<name>/history/chat/*.json`
- 完整日志: `users/<name>/history/log/*.json`
- 工具日志: `users/<name>/history/log/tool_log.json` (JSON)
- 归档: `users/<name>/history/archive/`
- 记忆: `users/<name>/agent_memory.db`
- 学习: `users/<name>/.learnings/`
