# 目录结构

最后更新：2026-05-04

## 顶层

```
votx-agent/
├── README.md                 用户说明
├── AGENTS.md                  编码 Agent 执行手册
├── 开发文档.md               开发文档总入口
├── start.py                  CLI / Web 入口
├── start_web.py              Web 专用入口
├── main.py                   CLI 主循环
├── requirements.txt          Python 依赖
├── pyproject.toml            pytest 配置
├── config/
│   ├── config_core.json      全局配置（历史/摘要/工具上限）
│   └── soul.md               全局人格基座，注入 system prompt
├── provider/
│   └── openai_api.py         DeepSeekProvider (流式/重试/Token/SSL)
├── run/
│   ├── engine.py             对话引擎 + system prompt 构建
│   ├── chat.py               ChatManager (历史/归档/修复)
│   ├── summarize.py          摘要 + 归档索引
│   └── tool.py               ToolRunner (注册/权限/限流/日志)
├── web/
│   ├── __init__.py
│   ├── server.py             Flask app + 错误处理 + 路由导入
│   ├── session.py            _session 状态 + _init_session
│   ├── commands.py           斜杠命令分发 + _web_summarize
│   ├── routes/
│   │   ├── chat.py           首页/用户/会话/SSE聊天/命令/断开
│   │   ├── files.py          上传/列表/下载/预览/删除
│   │   ├── conversations.py  归档列表/加载/删除/重命名
│   │   ├── system.py         消息/system prompt/导出/统计/工具日志
│   │   └── config.py         配置读写
│   └── templates/
│       └── index.html        单页前端（Vue 3）
├── skills/
│   ├── __init__.py           Skill 自动发现和注册
│   ├── _common/              通用工具 (安全/路径/截断)
│   │   # ── 工具型 (9) ──
│   ├── file/                 文件操作
│   ├── network/              网络请求
│   ├── shell/                Shell 执行
│   ├── time/                 时间查询
│   ├── video-download/       视频下载
│   ├── word-docx/            Word 文档
│   ├── tavily-search/        搜索
│   ├── uapi-hotboard-reporter/ 热榜
│   ├── ontology/             知识图谱
│   │   # ── 指令型 (11) ──
│   ├── self-improving/       自学习闭环
│   ├── vision/               图像识别
│   ├── download-anything/    资源下载
│   ├── file-search/          文件搜索
│   ├── find-skills/          Skill 查找
│   ├── pdf-tools/            PDF 操作
│   ├── skill-creator/        Skill 创建
│   ├── skill-vetter/         Skill 审查
│   ├── web-content-fetcher/  网页获取
│   ├── web-tools-guide/      网页指南
│   └── multi-user-long-term-memory/ 持久记忆
├── users/<name>/             用户运行数据
├── tools/skillhub/           SkillHub CLI
└── 开发文档/                 分章节开发文档
```

## Skill 结构约定

```
skills/<name>/SKILL.md        必需
skills/<name>/tool.py         工具型 Skill
skills/<name>/scripts/        可选脚本
skills/<name>/references/     可选参考文档
skills/<name>/assets/         可选资源
```

## 用户目录结构

```
users/<name>/
├── config.json               用户配置
├── self_soul.md              角色人设
├── agent_memory.db           持久记忆
├── .learnings/               学习记录
├── download/                 下载文件
└── history/
    ├── chat/                 对话历史
    ├── log/                  完整日志 + tool_log.json
    ├── archive/              归档对话
    └── file/                 上传文件
```
