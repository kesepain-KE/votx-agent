# 目录结构

```
kesepain-Agent/
├── start.py                  # 入口：用户选择 + 启动 main.py 子进程
├── main.py                   # 核心：主循环 + 命令分发 + Token 统计 + AGENT.md 注入
│
├── config/
│   ├── config_core.json      # 框架全局配置
│   └── soul.md               # 全局人设补充（HTML注释占位，非空时生效）
│
├── provider/
│   └── openai_api.py         # DeepSeekProvider — LLM 调用（流式+Token统计+重试）
│
├── run/
│   ├── chat.py               # ChatManager — 消息管理 + 历史压缩归档 + 清除/统计
│   └── tool.py               # ToolRunner — 注册/权限/限流/日志
│
├── skills/                   # Skill 标准骨架（agentskills.io 规范）
│   ├── __init__.py           # 自动发现：扫描 SKILL.md → 加载 tool.py → 注册
│   ├── _common/              # 公共模块包
│   │   └── __init__.py       # err()/truncate()/safe_path()/log_tool_call()
│   ├── file/                 # 文件工具
│   │   ├── SKILL.md
│   │   └── tool.py
│   ├── network/              # 网络工具（原 http.py，避免 stdlib 冲突）
│   │   ├── SKILL.md
│   │   └── tool.py
│   ├── shell/                # 命令工具（shell=False + 危险参数拦截）
│   │   ├── SKILL.md
│   │   └── tool.py
│   ├── time/                 # 时间工具
│   │   ├── SKILL.md
│   │   └── tool.py
│   ├── uapi-hotboard-reporter/  # 热榜查询
│   │   ├── SKILL.md
│   │   ├── tool.py
│   │   ├── scripts/          # 内部脚本（不被自动发现）
│   │   ├── references/       # 平台映射等参考文档
│   │   └── agents/           # 子 agent 配置
│   ├── video-download/       # 视频下载（yt-dlp 封装）
│   │   ├── SKILL.md
│   │   └── tool.py
│   ├── word-docx/            # Word 文档（python-docx 封装）
│   │   ├── SKILL.md
│   │   └── tool.py
│   └── tavily-search/        # Tavily 网络搜索
│       ├── SKILL.md
│       └── tool.py
│
├── agents/                   # [预留] Agent 角色定义
├── tools/                    # [预留] 独立 JSON Schema 文件
├── prompt/                   # [预留] Prompt 模板
├── AGENT.md                  # 项目指南（注入 system_prompt）
├── .env                      # 环境变量（API Key 等）
├── .env.example              # 环境变量模板
├── .gitignore
├── requirements.txt
└── 开发文档.md

└── users/
    ├── kesepain/
    │   ├── config.json       # Provider + 历史 + 工具权限
    │   ├── self_soul.md      # 角色灵魂（system prompt）
    │   └── history/
    │       ├── chat/          # 对话历史 JSON
    │       ├── log/           # 日志（chat_log.json + tool_log.json）
    │       ├── archive/       # 历史归档
    │       └── file/          # 关联文件
    │
    └── <username>/             # 更多用户按此模板创建
```
